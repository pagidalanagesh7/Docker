# 🐳 Docker Series – Day 4: Docker Volumes & Data Persistence

Welcome to **Day 4** of the Docker Learning Series!

In previous lessons, we learned how to build Docker images and run containers. However, containers are ephemeral, meaning the data inside them is temporary.

In this lesson, you'll learn how Docker stores persistent data using **Volumes**, **Bind Mounts**, and **tmpfs Mounts**.

---

# 📚 Topics Covered

- What is Data Persistence?
- Why Docker Volumes?
- Docker Volumes
- Named Volumes
- Anonymous Volumes
- Bind Mounts
- tmpfs Mounts
- Docker Volume Lifecycle
- Docker Volume Commands
- Backup & Restore Volumes
- Volume vs Bind Mount vs tmpfs
- Best Practices
- Interview Questions
